# Minha primeira página web

Esse projeto foi feito durante o curso **Minha Primeira Página Web (Front-End)** da PrograMaria.

A ideia foi aprender o básico de desenvolvimento web e colocar em prática criando uma página simples usando HTML, CSS e JavaScript.

## Tecnologias usadas
- HTML  
- CSS  
- JavaScript  

## Sobre o projeto
Aqui pratiquei estruturação de páginas, estilização com CSS e um pouco de interatividade com JavaScript.  
É um projeto de estudo, focado em aprendizado e evolução no front-end.
